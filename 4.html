编程环境配置

VS Code 插件

  1. Chinese：中文界面
  2. Prettier：代码美化
  3. Code Translate：翻译插件
  4. Live server：实时预览html
  5. Search/Translate Hero：搜索跳转和翻译
  6. vscode-icons：美化图标
  7. Git History Diff：后面git教程中出现的插件
  8. HTMLHint：检查html文件出错的地方

VS Code 配置

  1. Auto Save：自动保存，选择afterDelay
  2. Format On Save：勾选，保存后自动格式化
  3. Format On Paste：勾选，粘贴时自动格式化
  4. Default Formatter：选择prettier，默认格式化插件
  5. Linked Editing：勾选，同时重命名html标签名
  6. Accept Suggestion On Commit Character：取消勾选，不再自动补全一些乱七八糟的东西
  7. Accept Suggestion On Enter :选择off关闭，回车就是应该换行而不是自动补全乱七八糟的代码
  8. Tab Completion :选择on开启，如需自动补全，使用tab键

几个常用的快捷键

  • Ctrl + D：选中下一个匹配
  • Alt + 点击：插入光标
  • Ctrl + S：保存文件
  • Ctrl + Z：撤销
  • alt + Shift + 上下键：复制当前行
  • Ctrl + C：复制
  • Ctrl + V：粘贴
  • Ctrl + X：剪切
  • Ctrl + F：查找
